Features

Everything you need to ship AI with confidence

Regres.ai gives your team a complete regression testing workflow designed specifically for LLM-powered agents.

Replay Testing

Run your full baseline of historical inputs through new agent versions. Capture every output. Miss nothing.

Structured Diff Analysis

Not just text diffs. Field-level comparison with severity scoring: critical for decision fields, warnings for text changes.

Human Review Workflow

Approve expected changes, flag regressions, skip unknowns. Approved diffs automatically update your baseline.

Full Audit Trail

Every test run, every review decision, every change tracked with who, what, when. Built for compliance-heavy industries.

Pattern Detection

Automatically groups diffs that share common input attributes. Review 50 similar diffs with one click instead of one at a time.

API-First

Full REST API with API key auth. Trigger test runs from CI/CD pipelines, import baselines programmatically, export reports.

How It Works

Three steps to regression-proof your AI

Get from zero to full regression coverage in under 10 minutes.

1

Upload your baseline

Import your historical input/output pairs as JSON or CSV. These become the expected behavior your agent should match.

[{"input": {"claim_id": "CLM-001", ...}, "expected_output": {"decision": "approved", ...}}]
2

Connect your agent

Point Regres.ai at your agent's API endpoint. Configure headers, request templates, and response mappings. Test the connection with one click.

POST https://api.yourapp.com/agent/evaluate
3

Run, review, ship

Launch a test run. Review the diffs. Approve expected changes, flag regressions, and ship with confidence that nothing broke.
